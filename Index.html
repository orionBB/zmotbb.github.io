Guidelines:

- There must be a deck above a ramp.
- A deck that is used at the 2 ER level must be available for the entire season. You cannot remove it as buildings would be displaced.
- There has to be sufficient room for all defenses.
- Ramps 1&2 or 2&3 can be used at the same time. 1&3 is not a valid configuration.
